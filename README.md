# Lightgraf

Lightweight typography tool for Rails. It handles for you:
- __Quotes__:

    "Alpha", "Бета" and "Omega"\
    `↓`\
    “Alpha”, «Бета» and “Omega”
  
- __Hyphens__:

    Мой номер - +7 (977) 417-50-96\
    `↓` \
    Мой номер — +7 (977) 417-50-96

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'lightgraf'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install lightgraf
